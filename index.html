TESTS and HACKS

Test #1: Sampling Time for Proportional Controllers

The sampling times on python3 and micropython are different on ev3dev. I ran some tests on both Python3 and pybricks-micropython to see how fast they actually were. For this purpose I wrote a simple proportional controller code that would take ultrasonic sensor value and change the motor speed proportioanally.
